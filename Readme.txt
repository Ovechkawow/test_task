'.env'
Содержит переменные окружения, включая URL базы данных.
'database.py'
Определяет модели базы данных SQLAlchemy и настройку подключения.
'schemas.py'
Определяет модели Pydantic для валидации запросов и ответов.
'main.py'
Содержит приложение FastAPI

Запустить приложение:
"uvicorn main:app --reload"
